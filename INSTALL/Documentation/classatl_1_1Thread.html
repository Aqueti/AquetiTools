Detailed Description

A simple C++ wrapper class for POSIX threads.

! The intended usage of this class is via inheritence–it does nothing itself. First, create a new class that inherits from jthread. Overide the Setup and Execute methods. The Setup method is intended to be executed once. The Execute method is the actual thread (it contains the thread's execution loop, and does not exit except under special (i.e. fault) circumstances).
